## Introduction
File Machine is an open-source app to make backups of specific folders to your perfered location. Future releases will provide support for Windows and Linux, as well as a feature to schedule backups.

## Features
- Keeps Time Machine style backups of the changes you make. 
- Select as many folders as you want to backup

##  Prerequisites
MacOS 13.5 or higher(Windows and Linux versions coming soon)

## Installation
Download the appropriate file from the Releases folder

## Usage
Select the location where you backups will be stored. This can be on your computer or in a cloud or external drive.
